# Biso

A simple cat playground. Pick **Sea**, **Desert**, or **Grass**, then tap creatures as they zoom across the screen.

## Run

```bash
npx expo start
```

Then open in Expo Go, or press `i` / `a` for a simulator.

## Themes

| Theme  | Creatures                         |
|--------|-----------------------------------|
| Sea    | Fish, jellyfish, shrimp, crab     |
| Desert | Lizards, beetles, butterflies     |
| Grass  | Bunnies, birds, bees, squirrels   |

## Store builds

Use [EAS Build](https://docs.expo.dev/build/introduction/) with the included `eas.json` profiles (`development`, `preview`, `production`) when you are ready to ship to the App Store or Google Play.
